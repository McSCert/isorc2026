About ISORC 2026

The IEEE International Symposium on Real-Time Computing (ISORC) is a premier forum for researchers and practitioners to exchange ideas and present results on real-time computing systems, real-time applications, and enabling technologies.

ISORC 2026 will bring together experts from academia and industry to discuss the latest advances in real-time computing, including real-time operating systems, scheduling algorithms, quality of service, middleware, distributed systems, and embedded systems.

Topics of Interest

Real-Time Systems

  • Real-time scheduling and schedulability analysis
  • Real-time operating systems
  • Resource management

Embedded Systems

  • Embedded system design and implementation
  • Hardware/software co-design
  • System-on-chip architectures

Distributed Systems

  • Distributed real-time systems
  • Middleware for real-time applications
  • Cloud and edge computing

Applications

  • Cyber-physical systems
  • Internet of Things (IoT)
  • Autonomous systems and robotics
