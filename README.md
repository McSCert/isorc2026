# ISORC 2026

Official website for the IEEE International Symposium on Real-Time Computing 2026.

## About

ISORC is a premier forum for researchers and practitioners to exchange ideas and present results on real-time computing systems, real-time applications, and enabling technologies.

## Website

This repository hosts the GitHub Pages website for ISORC 2026. Visit the website at: https://mcscer.github.io/isorc2026

## Local Development

To run the website locally:

1. Install Jekyll: `gem install jekyll bundler`
2. Navigate to the repository directory
3. Run: `jekyll serve`
4. Open your browser to `http://localhost:4000`

Alternatively, you can simply open `index.html` in your browser to view the static site.

## Pages

- **Home**: Conference overview and topics
- **Call for Papers**: Submission guidelines and topics of interest
- **Important Dates**: Key deadlines and milestones
- **Committee**: Organizing and program committee members
- **Registration**: Conference registration information
- **Venue**: Location and travel information

## Contributing

To update the website, edit the HTML files in the repository and commit your changes. The site will be automatically updated via GitHub Pages.
